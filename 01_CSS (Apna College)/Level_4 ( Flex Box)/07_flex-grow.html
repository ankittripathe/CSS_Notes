flex-grow:- how much a flex item will grow relative to others if there’s extra space in the container.
  • Default: 0 (items won't grow).
  • If set to 1, the item will take up available space.
  • If multiple items have flex-grow, space is distributed proportionally.

    • flex-shrink:- how much a flex item will shrink relative to others if there’s not enough space in the container.
  • Default: 1 (items shrink to fit)
  • If set to 0, the item won't shrink (it will keep its size even if container is too small)
  • If multiple items have flex-shrink, shrinking is distributed proportionally
