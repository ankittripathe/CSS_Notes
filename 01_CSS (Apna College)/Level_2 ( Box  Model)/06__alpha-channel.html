Topic:- Alpha Channel
  • R => Red(0-255)
  • G => Green(0-255)
  • B => Blue(0-255)
  • A => Alpha[opacity (0-1)]
  • opacity set to 0 => no visible
  • opacity set to 0.5 => visible
  • opacity set to 1 => highly visible
  • Example of RGBA => color: rgba(255, 0, 0, 0.5)
