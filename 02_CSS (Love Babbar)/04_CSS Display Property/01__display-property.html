inline-block Inline-block elements are elements that have characteristics of both inline and block elements. They flow inline like inline elements, but you can apply block-level properties to them, such as setting width, height, margin, and padding. Here are some common HTML inline-block elements


  • img: Inline-block elements are commonly used for images.
  • button: Although inline by default, it behaves as an inline-block element when you apply styles like height and width.
  • input: The input element also behaves as an inline-block element.
  • label: Labels are often displayed inline but can be styled as blocks for layout purposes.
  • select: Dropdown select boxes behave as inline-block elements.
  • textarea: Similar to input, textarea elements are inline-block by default. These elements can be easily styled using CSS properties applicable to block-level elements while still retaining their inline flow characteristics.
