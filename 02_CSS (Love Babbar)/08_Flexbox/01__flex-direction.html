Flexbox=> It is one dimensional layout for arranging items in rows & columns

main-axes(horizental => x-axis) & cross-axes(vertically => Y-axis)

    flexbox direction

  • flex-direction: row; (default) [main axis = row (from left to rigt) i.e => horizentally & cross axis (top to buttom) i.e => vertically]
  • flex-direction: row-reverse; [main axis = row (from right to left) i.e => horizentally]
  • flex-direction: column; [main axis = column (from Top to bottom) i.e => vertically]
  • flex-direction: column-reverse; [main axis = column (from bottom to Top) i.e => vertically]
