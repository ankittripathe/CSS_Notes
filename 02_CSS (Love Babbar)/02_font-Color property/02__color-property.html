hsl(hue, saturtion, lightness). Hue is a degree on the color wheel from 0 to 360. ( 0 or 360) is red, 120 is green, 240 is blue.
Saturation can be described as the intensity of a color. It is a percentage value from 0% to 100%. 100% is full color, no shades of gray. 50% is 50% gray, but you can still see the color. 0% is completely gray, you can no longer see the color.
Lightness:- lightness of a color can be described as how much light you want to give the color,where 0% means no light(dark), 50% means 50% light (neither dark nor light), and 100% means full light.

hsla(hue, saturation, lightness, alpha). The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all).
