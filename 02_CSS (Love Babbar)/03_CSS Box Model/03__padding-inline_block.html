An element's padding-inline is the space from its border to its content in the inline direction(x-axis), and it is a shorthand property for the following properties: properties:
  • padding-inline-start (left-side)
  • padding-inline-end (right-side)
  • padding-inline: 10px 50px; padding at start is 10px padding at end is 50px
  • padding-inline: 10px; padding at start and end is 10px

    • An element's padding-block is the space from its border to its content in the block direction(y-axis), and it is a shorthand property for the following properties:
  • padding-block-start (top)
  • padding-block-end (bottom)
  • padding-block(shorthand): start(top) | end(bottom)
  • padding-block: 10px 50px; padding at start is 10px padding at end is 50px
  • padding-block: 10px; padding at start and end is 10px
