Padding-inline is the space from its border to its content in the inline direction (x-axis), and it is a shorthand property for the following Properties:-
  • padding-inline-start (left-side)
  • padding-inline-end (right-side)
  • padding-inline: 10px 50px; padding start = 10px, padding end = 50px
  • padding-inline: 10px; padding start and padding end = 10px

    • Padding-block is the space from its border to its content in the block direction (y-axis), and it is a shorthand property for the following Properties:-
  • padding-block-start (top)
  • padding-block-end (bottom)
  • padding-block(shorthand): start(top) | end(bottom)
  • padding-block: 10px 50px; padding start = 10px, padding end = 50px
  • padding-block: 10px; padding start and padding end = 10px
