Margin-inline:- property specifies the margin at the start(left) & end(right)in the inline direction (x-axis).
It is a shorthand property for the following Properties:
  • margin-inline-start (left-side)
  • margin-inline-end (right-side)
  • margin-inline: from both side

    • Margin-block:- property specifies the margin at the start(top) & end(bottom) in the block direction (y-axis).
    It is shorthand property for the following Properties:
  • margin-block-start (top-side)
  • margin-block-end (bottom-side)
  • margin-block: from top to bottom
  • margin-block: 10px; margin start and margin end is 10px
